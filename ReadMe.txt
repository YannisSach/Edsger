Σημασιολογικός αναλυτής της Edsger:
make : παράγεται το εκτελέσιμο compiler.
make clean: σβήνονται όλα τα ενδιάμεσα αρχεία που παρήχθησαν εκτός απο το εκτελέσιμο
make dstclean : σβήνεται και το εκτελέσιμο.
Ο compiler διαβάζει πλέον είτε από το standard input είτε απο αρχείο αν του δοθεί σχετική παράμετρος.

Σημασιολογοκοί έλεγχοι που εκτελούνται (μπορεί να υλοποιηθούν κι άλλοι μετά):
Έλεγχος για τα scope μεταβλητών, συναρτήσεων.
Έλεγχος για αντιστοιχία πραγματικών τυπικών παραμέτρων στις συναρτήσεις.
Έλεγχος για την ύπαρξη main.
Συμπερασμός τύπων έκφρασεων και έλεγχος αν εφαρμόζονται σωστοί τελεστές σε σωστούς τελεστέους.
Έλεγχος αν break και continue βρίσκονται μέσα σε κάποιο loop.

Επιπλέον έχει υλοποιηθεί συνάρτηση για την εκτύπωση του πηγαίου κώδικα όπως διαβάζεται από των parser.
Υποστηρίζονται include files τα οποία πρέπει να βρίσκονται στην ίδια θέση με το εκτελέσιμο (ακόμα κι αν πρόκειται για τις standard βιβλιοθήκες της Edsger)

Δεν υποστηρίζεται ακόμα η εμφάνιση γραμμής και στήλης στην οποία εντοπίστηκε λάθος κατα τη μεταγλώττιση.
