Χρησιμοποιήθηκαν οι εκδόσεις:
opam 4.03.0
llvm 3.5

Οδηγίες για μεταγλώττιση:
Αλλάζουμε τα paths στο Makefile.
Το makefile εκτελεί τις εντολές all clean και distclean όπως αυτές περιγράφονται στην εκφώνηση της άσκησης.
Το executable compiler αντιγράφεται και στο φάκελο Testcases.

Οδηγίες εκτέλεσης:
H εκτέλεση ./compiler [-f] [-i] [--ast] <source>.eds έχει αποτέλεσμα ένα αρχείο <source>.ll
Τα flags -f και -i έχουν την λειτουργία που περιγράφεται στην εκφώνηση.
Το flag --ast τυπώνει το AST.

Για πλήρη μεταγλώττιση:
Εκτελούμε το script Edsger.sh [-f] [-i] [-o] [--ast] <source>.eds με τη λειτουργία τους όπως έχει ήδη περιγραφεί.
